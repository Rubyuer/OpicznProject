# 朗速体育

  "朗速体育"是一款线上预约网球场馆和网球教练指导的应用。

  主要功能: 预约网球教练和场馆。   
  
    网球场馆下可直接预定网球场馆或者预定网球馆下的教练指导，含有教练指导的单价比普通余地网球场馆贵30/小时。
    
    各个网球场馆下都只配备了一名专业的网球教练，并且教练属于专职教练，专业性很强。
    
    预定成功的单子，需用户在预定时间内到达对应的网球场馆进行使用，超时未使用的单子自动取消。
    
    喜欢网球运动的小伙伴们，等你来撩---
